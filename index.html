ConCat App - Pearl Hacks 2020 Submission

Anonymous chat rooms for events and conferences, packaged in a lightweight Android app

This app is like Yik Yak but specifically for events. Enter an event room using a code given by the event organizers, and join the anonymous chat! Event organizers can easily set up a new chat room with a custom event code to share to participants. The app is built in Java, making use of Google Firebase Realtime Database as the backend. A peer and I built this from start to finish in less than 24 hours.
